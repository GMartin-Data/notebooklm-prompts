# Infographic — Vulgarisation/Concepts

**Source:** Custom - Validated
**Orientation:** Portrait (9:16)
**Detail Level:** Concise

## Steering Prompt

```
Flat design illustration, Kurzgesagt-style, soft pastel colors with warm accents.
Simplified rounded shapes, friendly icons, clean visual flow.
Use minimal text, rely on numbered steps and visual metaphors.
Structure as top-to-bottom progression: concept → mechanism → outcome.
```

## Variante Storybook

```
Educational infographic, children's storybook aesthetic, soft pastel palette.
Rounded shapes, simple icons, visual storytelling approach.
Numbered sections flowing vertically, one key idea per section.
Minimal labels only, let illustrations carry the explanation.
```

## Use Case (Data Engineering)

Expliquer des concepts techniques à des non-experts. Idéal pour :
- Introduction au Machine Learning pour stakeholders
- Vulgarisation de concepts cloud (serverless, containers)
- Onboarding non-technique sur les pipelines de données

**Test validé avec** : Ntiva - Machine Learning Beginner's Guide
