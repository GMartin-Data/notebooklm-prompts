# Essential Questions Generator

**Source:** XDA Developers (Steven Johnson)  
**Difficulty:** Beginner

## Prompt

```
1) Review all the uploaded lecture slides and generate five essential questions that, when answered, summarize the core concepts and key takeaways from the material.

2) For each question, identify which specific slides or sections contain the answer.

3) Answer each of the five questions in detail, making sure the answers clearly explain the underlying concepts from the slides.
```

## Use Case (Data Engineering)

Replace 'lecture slides' with 'ETL pipeline documentation' to extract core concepts from technical docs.
