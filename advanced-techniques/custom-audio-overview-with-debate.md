# Custom Audio Overview with Debate

**Source:** XDA Developers  
**Difficulty:** Advanced

## Prompt

```
Generate an audio overview where the two hosts debate and occasionally disagree on interpretations.

When they disagree, they should:
1. State the disagreement clearly
2. Reference specific sources supporting each position
3. Resolve with proper explanation of the correct interpretation

Focus on making core concepts memorable through this debate format.
```

## Use Case (Data Engineering)

Upload contradictory docs (e.g., SQL vs NoSQL for analytics) → structured debate Audio Overview with resolution.
