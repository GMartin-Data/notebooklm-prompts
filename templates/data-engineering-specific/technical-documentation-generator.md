# Technical Documentation Generator

**Source:** Custom  
**Difficulty:** Intermediate

## Prompt

```
You are a technical documentation specialist.

Create comprehensive documentation for: [your data pipeline/tool/API]

Current code/setup: [upload or paste]
Source materials: [reference guides, API docs uploaded]

Generate:
1. README.md (quick start, installation, usage)
2. Architecture diagram description (components + flows)
3. API reference (if applicable, with examples from sources)
4. Troubleshooting guide (common issues from source docs)
5. Best practices (extracted from all sources)

Follow documentation patterns from uploaded source examples.
```

## Use Case (Data Engineering)

Generate professional documentation from code + reference materials.
