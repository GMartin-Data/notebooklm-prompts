# Infographic — Social Media

**Source:** Custom - Validated
**Orientation:** Square (1:1)
**Detail Level:** Concise

## Steering Prompt (Cyberpunk — Recommandé pour tech/IA)

```
Tech trend infographic, cyberpunk-inspired dark theme.
Dark background with neon teal and magenta accents.
One powerful statistic or prediction prominently displayed.
Glowing effects on key elements, futuristic sans-serif font.
Designed for instant comprehension, social media optimized.
```

## Variante Gradient (Corporate/Neutre)

```
Bold statement infographic, eye-catching modern aesthetic.
Vibrant gradient background transitioning from deep purple to electric blue.
Single hero statistic or insight as focal point, large bold typography.
Minimal supporting text, high visual impact for scroll-stopping effect.
Clean iconography, no clutter.
```

## Use Case (Data Engineering)

Créer du contenu viral pour réseaux sociaux. Idéal pour :
- Stats marquantes sur l'IA/ML
- Tendances tech à partager
- Insights data pour LinkedIn/Twitter

**Test validé avec** : Our World in Data - AI

**Note** : La variante Cyberpunk offre un meilleur contraste et impact visuel que la variante Gradient pour le contenu tech.
