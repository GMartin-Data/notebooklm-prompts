## Style visuel

Coller dans le champ "Describe a custom visual style" :

```text
3D isometric illustration, corporate tech aesthetic, soft ambient lighting, smooth surfaces, subtle gradients, modern enterprise feel.
Depict process as a timeline with milestones and decision points.
```

## Focus IA

Coller dans le champ "What should the AI hosts focus on" :

```text
Target audience: [AUDIENCE] evaluating [PRODUCT/FEATURE]

Focus on:
- Lead with the problem being solved
- Showcase key capabilities with concrete benefits
- Emphasize what's new or different vs alternatives
- Use real-world scenarios to illustrate value
- End with clear next steps or call to action

Skip:
- Technical implementation details
- Exhaustive feature lists
- Pricing or licensing specifics
- Known limitations (unless critical)

Tone: Confident, forward-looking, benefit-oriented
```
